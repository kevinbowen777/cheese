Installation
============

Cloning the project
-------------------

To install the ``cheese`` project, run the following command in your terminal:

.. code-block:: console

   $ git clone https://github.com/kevinbowen777/cheese.git
   $ cd cheese

Depending on your needs and preferences, see either the
:doc:`local_installation` or :doc:`docker_installation` page for additional
installation instructions.
